Project Overview

This is an End-to-End Data Analytics Project developed using Python, MySQL, and Power BI to analyze customer shopping behavior and generate business insights.

The project covers the complete analytics workflow including:

Data Cleaning
Feature Engineering
SQL-based Analysis
Interactive Dashboard Creation
Technologies Used
Python
Pandas
NumPy
MySQL
Power BI
Jupyter Notebook
Project Workflow
1. Data Preprocessing using Python

Performed data cleaning and preprocessing tasks such as:

Handling missing values
Renaming columns
Feature engineering
Creating age groups
Mapping purchase frequency values
Removing unnecessary columns
2. SQL Analysis

Processed data was connected to MySQL for analytical querying and insight generation.

Performed:

Customer segmentation
Purchase analysis
Revenue insights
Customer behavior analysis
Frequency-based analysis
3. Power BI Dashboard

Created an interactive dashboard to visualize:

Customer spending patterns
Purchase trends
Category-wise analysis
Customer segmentation insights
Business KPIs
